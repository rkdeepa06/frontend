About us:

CodeCrest is a competitive programming platform where students, problem setters, and admins can manage coding challenges, contests, and rankings. It helps contestants practice algorithmic problems and track their progress on a global leaderboard.

Users can solve coding challenges of different difficulty levels and submit their source code for evaluation. Each submission gets an instant verdict such as Accepted or Wrong Answer, along with points based on difficulty and time taken.

One of the key features of CodeCrest is its contest system. Users can enroll in scheduled programming contests within a fixed time window and compete with other contestants in real time.

The platform also includes a leaderboard that shows the global rank of every contestant based on total points, as well as a contest-specific leaderboard based on score and penalty time.
